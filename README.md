# Human-Resources-Analytics-Dashboard
A Power BI dashboard project for analyzing human resources data and presenting workforce insights through interactive visualizations.


## 📌 Project Overview
The dashboard focuses on presenting HR insights in a clear and interactive way to support better understanding and decision-making.

## 🎯 Problem Statement
Organizations manage large amounts of employee-related data, and HR teams often face challenges in analyzing this data effectively and presenting insights in a clear visual format.

## 📈 Visualizations Used
- Pie Chart
- Donut Chart
- Cards
- Slicer
- Bar Chart
- Gauge Chart
- Stacked Column Chart
- Table
- Funnel Chart

## 🛠️ Tools Used
- Microsoft Power BI
- Data Visualization
- Dashboard Design

## 💡 Skills Demonstrated
- Data Analysis
- Data Visualization
- Dashboard Development
- Business Insight Presentation
- HR Data Reporting

## 🚧 Challenges
One of the main challenges in this project was limited time and resources, which affected the ability to build additional visuals such as hierarchy charts.

## ✅ Conclusion
This dashboard highlights the importance of HR analytics in understanding workforce-related data and presenting insights through effective visual storytelling.
